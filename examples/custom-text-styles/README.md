# Custom Text Styles Example

Customize the appearance of text overlays in your videos.

## Purpose

Demonstrates how to configure text styling options — font size, color, position, and animation — for subtitles and title cards.

## Expected Output

A video with styled text overlays that match the specified configuration, demonstrating customization beyond defaults.

## Usage

```bash
# From the project root
cp examples/custom-text-styles/input-scripts.json input/
npm run generate
```

## Customization Options

Text styling can be configured in `input-scripts.json` or via environment variables. See the full [configuration reference](../docs/configuration.md) for available options.
