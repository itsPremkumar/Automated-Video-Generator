# Examples

This directory contains practical examples for using Automated Video Generator.

## Available Examples

| Example | Description |
|---------|-------------|
| [youtube-shorts](./youtube-shorts/) | Generate YouTube Shorts-style portrait videos |
| [faceless-channel](./faceless-channel/) | Faceless YouTube channel automation |
| [local-assets](./local-assets/) | Using your own images and videos |
| [batch-processing](./batch-processing/) | Generate multiple videos in one run |
| [director-mode](./director-mode/) | Using `[Visual:]` tags for scene control |
| [mcp-agent](./mcp-agent/) | Using the MCP server with AI agents |
| [multi-language](./multi-language/) | Videos in Tamil, Hindi, Spanish, and more |
| [background-music](./background-music/) | Adding music with auto-ducking |
| [custom-text-styles](./custom-text-styles/) | Custom text configuration |

## How to Run Examples

1. Copy the example's `input-scripts.json` to the project root's `input/` directory
2. Ensure your `.env` is configured
3. Run `npm run generate`

Or use individual script files directly with the CLI.
