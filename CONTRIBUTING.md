# Contributing to Sproutern Video Generator

First off, thank you for considering contributing to the Sproutern Video Generator! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue with:
- A clear title and description.
- Steps to reproduce the bug.
- Any relevant logs or screenshots.

### Suggesting Enhancements
Have a great idea? Open an issue and describe:
- What the enhancement is.
- Why it would be useful.
- How you imagine it working.

### Pull Requests
Ready to code?
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the existing style.
5. Issue that pull request!

## Pull Request Guidelines
- We use [Prettier](https://prettier.io/) for code formatting.
- Please keep your PRs focused. If you have several unrelated changes, submit them as separate PRs.
- Write clear, descriptive commit messages.
