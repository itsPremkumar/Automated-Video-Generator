# Supported Versions

This project follows **semantic versioning** and maintains the following release support policy:

| Version | Status              | Support Level        |
| ------- | ------------------- | -------------------- |
| 5.x     | Active development  | ✅ Full support      |
| 4.x     | Maintenance         | ⚠️ Security fixes    |
| < 4.0   | End of life         | ❌ No support        |

## What "Full Support" means

- Bug fixes and security patches
- New features and improvements
- Documentation updates
- Compatibility with latest dependencies

## What "Maintenance" means

- Critical security patches only
- No new features
- Limited documentation updates

## Upgrading

We strongly recommend using the latest version. See [CHANGELOG.md](./CHANGELOG.md) for upgrade notes.
