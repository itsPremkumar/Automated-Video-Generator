# Roadmap

## Current Focus (v5.x)

- [x] Windows desktop app with bundled runtime
- [x] MCP server for AI agent integration
- [x] AI visual media verification
- [x] Free stock media (Openverse, Wikimedia Commons, Internet Archive)
- [x] Local voice synthesis providers (Voicebox, XTTS, Kokoro)
- [x] Background music with auto-ducking

## Short-term (v5.5 - v6.0)

- [ ] Comprehensive test suite (unit + integration + E2E)
- [ ] Plugin system for custom media sources
- [ ] Custom subtitle styling (fonts, positions, animations)
- [ ] Template system for reusable video styles
- [ ] Performance benchmarks and optimization report
- [ ] macOS and Linux desktop builds
- [ ] API documentation with Swagger/OpenAPI

## Medium-term (v6.0 - v7.0)

- [ ] Visual timeline editor in the web portal
- [ ] Real-time collaboration features
- [ ] Advanced scene transitions (crossfade, wipe, morph)
- [ ] AI script generation with GPT/Llama integration
- [ ] Multi-track audio (voiceover + music + sound effects)
- [ ] Custom video resolution and aspect ratio presets
- [ ] Export presets for YouTube, TikTok, Instagram
- [ ] Headless API mode for programmatic access

## Long-term (v7.0+)

- [ ] Native GPU acceleration for rendering
- [ ] Cloud rendering queue with distributed workers
- [ ] Template marketplace for community sharing
- [ ] AI-powered storyboard generation from prompts
- [ ] Real-time preview during rendering
- [ ] Mobile companion app for remote monitoring
- [ ] Enterprise SSO and team management

## Vision

Automated Video Generator aims to become the **open-source standard** for programmatic video generation — the way WordPress is for websites and OBS is for streaming. We want to make high-quality video production accessible to every developer, creator, and organization.
